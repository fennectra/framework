V Quick Reference

Essential CLI Commands 
# Dev server
./forge serve                        # PHP built-in
./forge serve --frankenphp --worker  # FrankenPHP worker (max perf)

# CRUD generation
./forge make:all Product --roles=admin,manager
./forge make:all Invoice --connection=job

# Migrations & Seeding
./forge migrate                        # apply migrations
./forge migrate --rollback             # rollback the last batch
./forge make:migration create_products # create a migration
./forge db:seed                        # run the seeders

# Queue & Scheduler
./forge queue:work                     # consume jobs
./forge queue:retry --id=5             # re-execute a job
./forge schedule:run                   # scheduled tasks

# Feature Flags & Deploy
./forge feature list                   # list flags
./forge feature enable dark-mode       # enable a flag
./forge deploy                         # build + push + K8s

# Quality
./forge quality        # lint + PHPStan + tests
./forge quality --fix  # auto-fix style
Project Structure 
src/                    ← Framework core
  Attributes/           ← Validation, ORM, RateLimit, StateMachine, Broadcast
  Commands/             ← CLI (serve, make:*, migrate, queue, schedule, deploy)
  Core/                 ← App, Router, Container, ORM, Events, JWT
    Profiler/           ← Debug profiler per-request
    Relations/          ← Eager loading (BelongsTo, HasMany, HasOne)
    Cache/              ← RedisCache, TaggedCache
    Scheduler/          ← Cron + K8s-safe Redis Lock
    Queue/              ← Job dispatch + worker
    Feature/            ← Feature flags
    Notification/       ← Mail, Slack, Database
    Broadcasting/       ← SSE via Redis Pub/Sub
    OAuth/              ← Google, GitHub
  Middleware/           ← Auth, CORS, Profiler, RateLimit, Security, Logging

app/                    ← Application code
  Controllers/          ← HTTP handlers
  Models/               ← ORM models
  Dto/<Domain>/         ← DTOs organized by domain
  Routes/               ← Route files (auto-loaded)
  Jobs/                 ← Job classes for the queue
  Schedule.php          ← Scheduled tasks

database/               ← Schema and data
  migrations/           ← Migration files
  seeders/              ← Seeders

docker-compose.yml      ← Local dev (API + PostgreSQL + Redis)
HTTP Architecture
HTTP Request │ ▼ index.php ──► App::run() or App::runWorker() │ ▼ Router::dispatch() │ ▼ MiddlewarePipeline │ ├── CorsMiddleware (CORS headers) ├── ProfilerMiddleware (per-request debug) ├── RequestLoggingMiddleware (logs + request_id) ├── SecurityHeadersMiddleware(security headers) ├── RateLimitMiddleware (429 if exceeded) ├── AuthMiddleware (JWT + RBAC) │ ▼ Controller::method() │ ├── Model / DB (ORM + eager loading) ├── Validator / DTO (validation) ├── Event::dispatch() (events + notifications) ├── Job::dispatch() (async queue) ├── FeatureFlag::enabled() (rollout) │ ▼ JSON Response
